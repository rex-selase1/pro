HOW TO USE COMMAND PROMPT TO FORMAT A DRIVE USING DISKPART.

By Kugblenu shamouah added on 8th September, 2023.

Command prompt is a command shell program used to accomplish so many tasks on the computer. It is basically a text-based UI (User Interface) that is used to run commands and performs advanced administrative functions.It can be used to create folders and directories and can be used to move files around in the storage.It is available in every Windows operating system.

How the command prompt interface looks.

ACCESSING THE COMMAND SHELL..

Command Prompt can be accessed by two very popular means. You can use the Windows Search bar or a Windows Run command. You can find the Windows Search bar in Tablet mode or When you hide your Taskbar by pressing the Windows Button on your Keyboard .

Using Windows search bar to access command prompt.

To access Command Prompt via Windows Search bar, Press the start or Windows Button on your Keyboard or Use your mouse to click on the Windows Search bar and type, "Command prompt" and hit Enter Button. This opens the Command Prompt Window.

Using the Windows Run Shortcut.

To use the Windows Run Shortcut, Press the following keys : (ctrl + R ) together. The command window will open with a dialogue box. Type cmd in the box and hit Enter. This opens the Command Prompt Window.
Type, DISKPART in the Text field and hit Enter. I recommend when launching Command Prompt, You should open it with administrative privileges or Else when you type Diskpart and press enter, Windows will ask you to give administrative privileges by clicking yes on the pop-up prompt window.

Type diskpart and press Enter

After typing Diskpart and hitting enter,A seperate window opens. That's the diskpart Window.

Diskpart window

List Disk.

Type (List Disk), and hit enter. This command lists all the disks inside your system or connected externally.
